# NammaMistri

NammaMistri is a bilingual English and Kannada construction management Android app for masons and contractors.

## Features

- Project management with add, edit, search and confirmed cascade delete
- Worker management per project
- Attendance insert/update per worker
- Expense tracking per project
- Local photo URI storage per project
- Summary dashboard with project progress, readable creation date, labour cost, expense total and final total
- In-app English/Kannada language switch across the Compose UI

## Tech Stack

- Kotlin
- Jetpack Compose Material3
- Room Database
- Kotlin Coroutines

Open this folder in Android Studio, sync Gradle, and run the `app` configuration.
